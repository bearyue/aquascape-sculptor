# Aquascape Sculptor

一个基于 Three.js 的鱼缸洞穴造景 3D 雕刻原型工具。

当前目标是先做一个可交互的 Web 原型：从一块带厚度的实体地台开始，通过鼠标笔刷、框选、开洞等操作，自定义鱼缸造景起伏，并导出 STL 文件用于 3D 打印验证。

## 当前功能

- 初始生成带厚度的矩形地台
- 支持调整地台尺寸：宽度、深度、基础厚度
- 支持笔刷编辑：
  - 抬高
  - 压低
  - 平滑
- 支持框选区域：
  - 选区抬高
  - 选区压低
  - 选区平滑
- 支持点击开圆形通孔
- 支持一键生成自然起伏
- 支持撤销、重做、重置
- 支持导出 STL
- 页面内置基础自检

## 本地运行

由于页面使用了 ES Module 和 importmap，不建议直接双击打开 HTML。

推荐使用本地 HTTP 服务运行：

```bash
python -m http.server 8080
```

然后浏览器打开：

```text
http://localhost:8080
```

也可以使用 VS Code 的 Live Server 插件运行。

## 使用方式

1. 打开页面后，默认是“浏览旋转”模式，可以拖动视角查看模型。
2. 选择“抬高 / 压低 / 平滑”，在模型上按住鼠标左键拖动，即可雕刻地形。
3. 选择“框选”，拖出矩形区域后，可以对选区执行抬高、压低、平滑。
4. 选择“开圆洞”，点击模型表面即可开一个垂直通孔。
5. 点击“导出 STL”，得到可用于切片软件的 STL 文件。

## 3D 打印建议

当前版本主要用于造型验证，真实打印前建议在切片软件中检查：

- 是否需要支撑
- 洞口边缘是否过薄
- 是否存在悬垂过大的区域
- 底部是否足够平稳
- 模型尺寸是否适合鱼缸内部空间

材料建议优先考虑 PETG。PLA 长期泡水稳定性较差，不建议用于长期鱼缸环境。
